User Stories:

1* As a user, I want to be able to read all items so that I am able to decide what I want to get from a list.

2* As a user, I want to be able to create an item so that I am able to make a list.

3* As a user, I want to be able to delete items so that I am able to remove from a list.

4* As a user, I want to be able to edit items so that I am able to change from a list.

5* As a user, I want to be able to search for an item in the database.

6* As a user, I want to have reasonable results for my search in the database.

7* As a user, I want to click on an item, and get more information about it.

8* As a user, I want to be able to send feedback to the website, or ask questions.

9* As a user, I want to be able to link to social media sites from the web page.

10* As a user, I want to navigate easily between my lists so that I can personalize my space easily.
